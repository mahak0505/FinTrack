Trust & Safety

Security at Finly

Your financial data is deeply personal. Here's how we protect it.

🔐 Encryption

All data stored on Finly servers is encrypted at rest using AES-256 — the same standard used by banks. All data transmitted between your device and our servers is encrypted using TLS 1.3, the latest transport security protocol.

🏗️ Infrastructure

Finly's backend runs on SOC-2 compliant cloud infrastructure. We use isolated environments, regular vulnerability scans, and automated security monitoring. Our databases are never publicly accessible.

🔑 Authentication
  • Passwords are hashed using bcrypt with individual salts — we never store plain-text passwords
  • Session tokens are rotated on each login and expire automatically
  • Rate limiting prevents brute-force attacks on login endpoints
  • Two-factor authentication (2FA) is on our near-term roadmap
🔍 What We Don't Collect
  • Bank account numbers or login credentials
  • Credit or debit card details
  • Government IDs or Aadhaar information
  • Biometric data of any kind
📣 Reporting a Vulnerability

If you discover a security issue in Finly, please report it responsibly to security@finly.in. We take all reports seriously and will respond within 48 hours. We appreciate responsible disclosure and credit researchers who help us improve.
